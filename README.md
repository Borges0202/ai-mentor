# AI Mentor - Learning Project

## 📌 Sobre o projeto

Este projeto faz parte do meu processo de aprendizado em desenvolvimento utilizando **JavaScript** e **Node.js**.

O objetivo é criar um **agente mentor de programação** que ajude o usuário a aprender conceitos de programação sem fornecer imediatamente a solução completa.  
A ideia é incentivar o raciocínio, oferecendo dicas, explicações e direcionamentos.

Este repositório também funciona como um **diário de aprendizado**, onde registro minha evolução enquanto estudo conceitos de backend e interação com terminal.

---

## 🎯 Objetivos do projeto

- Praticar **JavaScript fora do navegador**
- Aprender a usar **Node.js**
- Criar programas interativos no terminal
- Entender conceitos iniciais de **agentes de IA**
- Desenvolver boas práticas de organização de projeto

---

## ⚙️ Tecnologias utilizadas

- JavaScript
- Node.js
- Terminal / CLI
